                Data structures are used to store and manage data in computer 
                science in a way to be efficiently retrieved. Some are simple and
                for general problems and then there are some that are more 
                specialized. Some data structures are better if traversed 
                iteratively and others are more efficient if traversed recursively.

There are many different kinds of data structures, some examples are:

  • Set
  • Stack
  • Queue
  • Linked List
  • Map
  • Tree
